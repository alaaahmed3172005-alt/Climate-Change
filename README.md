# Climate-Change
This project is a motion graphics video that highlights the issue of climate change. It explains its causes and effects on the environment and humans through simple and engaging visuals. The video uses animated graphics and sound to deliver an awareness message that encourages protecting the environment and reducing harmful practices.

> 📎 Project Name : Climate Change
----------------------
> 📌 Project Overview: This project is a motion graphics video that highlights the issue of climate change. It explains its causes and effects on the environment and humans through simple and engaging visuals. The video uses animated graphics and sound to deliver an awareness message that encourages protecting the environment and reducing harmful practices.

> 🎯 Project Objectives:
 - Raise awareness about climate change
 - Present information in a clear visual way
 - Develop the team’s motion graphics and collaboration skills
 - Produce a video suitable for sharing on social media
----------------------
> 📦 Project Scope :
 The project includes idea development, script writing, visual design, animation production using motion graphics tools, sound effects integration, and exporting a short final video. It does not include long-form cinematic production or advanced documentary content.
📅 Project Plan (5 Weeks)

Week 1
Research the topic and write the concept and script

Week 2
Design characters and visual elements

Week 3
Animation production

Week 4
Add sound, editing, and review

Week 5
Final adjustments, export, and presentation
